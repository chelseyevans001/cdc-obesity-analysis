# CDC Obesity Data Analysis

## Project Overview
This project analyzes public health data from the CDC PLACES API to identify patterns in obesity prevalence across U.S. states and counties.

## Tools Used
- Python
- Pandas
- Matplotlib
- CDC Public Health API
- Google Colab

## Key Steps
1. Retrieved public health data from the CDC API
2. Cleaned and prepared the dataset
3. Filtered records related to obesity prevalence
4. Analyzed obesity rates across U.S. states and counties
5. Created visualizations to identify geographic patterns

## Key Insights
- Some states exhibit significantly higher obesity prevalence compared to others.
- County-level analysis reveals local variations in obesity rates within states.
- Data visualization helps highlight regions where public health interventions may be most needed.
